# jc-datetime
A Javascript/Node datetime tool.

# 1. 安装

可以使用 npm 进行安装：

```shell
npm install jc-datetime
```
或者使用 yarn 进行安装：

```shell
yarn add jc-datetime
```

# 2. 使用教程

该模块提供了 `Second`, Minute`, Hour`, Date_`, `DateTime` 五个对象，分别可以用于 秒、分、时、日期、日期时间的处理。
